# Проект Место-React (frontend)
### Место - это интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки. 
### Данная работа выполнена на основе таких технологий как:
* React
* React Router
* Nested BEM
* JSX
* Адаптивная верстка
